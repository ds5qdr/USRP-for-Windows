# USRP-for-Windows
DVSwitch USRP Client modified version 2.54

# How to setup
- download USRP-for-Windows-main.zip
- unzip
- edit usrp.ini
- run USRP.exe
- for more information, click https://ds5qdr-dv.tistory.com/214
